Avia

Блог с заметками об изучении тем, связанных с авиацией.

Quick Start

1. Скачать файлы с github. Нажать на зеленую кнопку Code и скопировать HTTPS
2. В терминале перейти в папку, где будет храниться проект
3. В терминале набрать команду git clone и скопированный HTTPS
4. Настроить виртуальное окружение командой uv sync
5. Запустить сервер через команду make run.
